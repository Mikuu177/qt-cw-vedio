# Ethics Documentation

## Overview
This folder contains all ethics compliance documentation for user evaluations.

## Required Documents
- `informed_consent_template.md` - Consent form for participants
- `participant_information_sheet.md` - Information about the study
- `data_privacy_protocol.md` - How data will be handled
- `anonymization_procedures.md` - How participant identity is protected

## Important Notes
- All participants must sign consent forms before evaluation
- Personal data must be anonymized in reports
- Data should be stored securely and deleted after assessment
- Comply with GDPR and university ethics guidelines
