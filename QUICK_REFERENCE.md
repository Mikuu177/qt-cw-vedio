## ⌨️ 应用快捷键速查表

### 播放控制
- **Space**: 播放/暂停
- **←/→**: 快进/快退 5 秒
- **↑/↓**: 增加/减少音量
- **M**: 静音

### 编辑功能
- **I**: 设置入点 (In Point)
- **O**: 设置出点 (Out Point)
- **M**: 添加标记
- **[**: 跳转到上一个标记
- **]**: 跳转到下一个标记
- **Ctrl+Z**: 撤销
- **Ctrl+Y**: 重做

### 导出和视图
- **Ctrl+E**: 打开导出对话框
- **Ctrl+Shift+H**: 切换高对比度模式
- **F**: 全屏模式
- **F1**: 打开帮助对话框
- **Ctrl+O**: 打开视频文件
- **Ctrl+Q**: 退出应用

- **质量**: 1080p 清晰度

- **长度**: 1 分 - 1 分 30 秒
- **内容**: 完整功能演示，面向非技术观众
- **字幕**: 中文或英文
- **质量**: 1080p 清晰度

- **录屏**: OBS Studio (免费)
- **编辑**: DaVinci Resolve (免费)

### 常见问题

**Q: 应用启动失败？**  
A: 检查 Python 版本 (需要 3.8+)，运行 `pip install -r requirements.txt`

**Q: 添加片段时长不准确？**  
A: 已在 2025-11-26 修复，确保使用最新代码

**Q: 导出功能不可用？**  
A: 需要安装 FFmpeg，或使用其他功能 (不需要 FFmpeg)

**Q: 高对比度模式不生效？**  
A: 按 Ctrl+Shift+H，或从菜单 View > Toggle High Contrast Mode
